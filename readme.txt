== Description ==
Cude Blog is a simple WordPress theme for the blogging and news site. Lightweight and elegant design will make your web project irresistible.

* Google Mobile Friendly & Google Structured Data Tested
* Font awesome support
* SEO Optimized
* Cross browser support
* Mobile adaptive
* Simple and Friendly theme options panel
* WPML support
* Multisite support

== Installation ==

1. Copy the theme folder to /wp-content/themes/
2. Activate the theme
